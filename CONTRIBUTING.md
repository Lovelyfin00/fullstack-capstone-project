# Contributing to this Project

Thank you for investing your time in contributing to our project! 

## How to Contribute

### Reporting Bugs
Before creating bug reports, please check the existing issues to see if the problem has already been reported. When creating an issue, include as many details as possible to help us reproduce the bug.

### Pull Requests
1. Fork the repository and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Open a Pull Request with a clear title and description of your changes.
